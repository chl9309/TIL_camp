## 1. 숫자의 입력과 출력



[입력]
숫자를 2번 입력받는다.


[출력]
입력 숫자를 계산하여 값을 출력한다.


[입력 예시]
6374
8729


[출력 예시]
15103



```python
x = int(input())
y = int(input())

print(x + y)
```
