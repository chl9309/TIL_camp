## Git 기본기

- working directory
    내가 작업하고 있는 실제 디렉토리

- staging area
    커밋으로 남기고 싶은,
    특정 버전으로 관리하고 싶은 파일이 있는 곳
    작업 한 것들 변경사항이 임시로 저장 되는 곳

- repository
    커밋이 저장되는 곳


- github 등에 연결 하여 저장



#### 사용법

 - git add
        - git

 - git commit


 - git push -new orgin master URL


 - git clone URL
        - git

 - git pull
